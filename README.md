## Run Locally

**Prerequisites:**  [Android Studio](https://developer.android.com/studio)


1. Open Android Studio
2. Select **Open** and choose the directory containing this project
3. Allow Android Studio to fix any incompatibilities as it imports the project.
4. Create a file named `.env` in the project directory and set `GEMINI_API_KEY` in that file to your Gemini API key (see `.env.example` for an example)
5. Remove this line from the app's `build.gradle.kts` file: `signingConfig = signingConfigs.getByName("debugConfig")`
6. Run the app on an emulator or physical device

# 📱 Download & Install the Last Minute Life Saver APK

Thank you for taking the time to evaluate **Last Minute Life Saver**!

To try the application on your Android device, please follow the steps below:

## 1. Download the APK

* Navigate to the **Releases** section of this GitHub repository (or open the APK file included in the repository).
* Download the latest `LastMinuteLifeSaver.apk` file to your Android device.

## 2. Enable Installation from Unknown Sources

Since this APK is being distributed through GitHub and not the Google Play Store, Android requires permission to install apps from external sources.

* Open **Settings** on your Android device.
* Go to **Security & Privacy** (or **Apps > Special App Access > Install Unknown Apps**, depending on your Android version).
* Enable **Install from Unknown Sources** or **Allow from this source** for the browser or file manager you used to download the APK.

> **Note:** This permission is only required because the app is being installed outside of the Google Play Store.

## 3. Install the App

* Open your **Downloads** folder.
* Tap the downloaded `LastMinuteLifeSaver.apk`.
* Select **Install** and wait for the installation to complete.

## 4. Launch the App

Once the installation is complete:

* Tap **Open**, or
* Find **Last Minute Life Saver** in your app drawer and launch it.
